- Preferred branch: `project/andromeda`
- Intended workflow: safe for Claude, Codex, or any IDE as long as the branch stays project-specific.

Suggested next folders when you start building:

- `app/`
- `docs/`
- `assets/`

For now this folder is a clean landing zone so you can start committing without mixing CRM work into other projects.
