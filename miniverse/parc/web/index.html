parc

Agent-first architecture intelligence

$ npx parc map ./your-repo

Drop any git repo. Get an interactive architecture map.

How it works
1
Drop a repo
Point parc at any git repo on your machine. No config files, no setup, no dependencies to install.
2
Instant analysis
parc scans your codebase in seconds. Routes, APIs, agents, components, database schema — all detected automatically.
3
Interactive diagrams
Get a single HTML file you can open, share, or host. Searchable, filterable, and always up to date.
Commands
parc map <repo>
Architecture Map
Scan any git repo and generate an interactive Gravity-styled HTML diagram. Detects routes, APIs, AI agents, components, database tables, and dependencies.
parc serve <repo>
Live Dashboard
Start a dark-mode live dashboard for your repo. Real-time stats, agent visualization, and a rescan button. Perfect for team standups.
parc agents <repo>
Agent Registry
List every AI agent in your codebase with model tier detection, tools used, and descriptions. Know exactly what your agents do.
parc diff [ref]
Architecture Diff
See how your architecture changed since any commit. Shows added routes, modified agents, new API endpoints, and changed components.
What it detects
Frameworks
Next.js, Vite, SvelteKit, Express
Auto-detects your framework, version, and features like Tailwind, Supabase, Prisma, Zod.
Routes & APIs
App Router, Pages, REST
Maps every page route and API endpoint with HTTP methods and auth detection.
AI Agents
Model tiers & tools
Finds agents by file pattern, detects reasoning vs extraction tiers, and lists tools each agent uses.
Components
UI & Feature split
Catalogs your component library, separating base UI components from feature-level ones.
Database
Tables & Migrations
Reads Prisma schemas and SQL migrations to map your data model and migration history.
Git
Branch, commits, remote
Shows current branch, last commit, author, commit count, and remote URL at a glance.

  • Grep through code to find all API endpoints
